# 第一章 基础
## 1.1基本逻辑与复合逻辑
### 1.1.1 基本逻辑：
与  A.B
或  A+B
非  !A

### 1.1.2 复合逻辑：
与非（先与再非）：
$$Y = \overline{A \cdot B}$$
或非（先或再非）：
$$Y = \overline{A + B}$$
异或（选不同的为1）：A $\oplus$ B
同或（选相同的为1）：A $\odot$ B
